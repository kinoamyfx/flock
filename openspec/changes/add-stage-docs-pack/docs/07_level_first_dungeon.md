# Level｜首个秘境切片（关卡口径）

## Outcome
给“可玩闭环”提供可回归的关卡节奏与投放口径（不追求内容量，只追求可验证）。

## Target（v1.0.0 最小）
- 目标：玩家能在 3–8 分钟内完成一次回合（撤离或死亡）。
- 风险：全掉落带来挫败 → 用“安全箱/保护期/清晰提示”降低无谓损失。

## Beats（节奏分段）
1) 入口区（30s）：教学提示（移动/拾取/撤离）
2) 低风险区（2min）：少量资源掉落，鼓励拾取与背包操作
3) 冲突区（2–4min）：引入“死亡/掉落/保护期”可见反馈（可用 debug kill/self）
4) 撤离点（30s）：读条/可被打断反馈

## Spawn & Drops（投放口径）
- MVP：掉落表可占位，但必须具备唯一 lootId 与保护期信息展示。

## Failure Conditions（失败条件）
- 死亡：携带物掉落（安全箱不掉落）
- 断线：留身 60s 可被击杀（同上）

## Acceptance
- 通过 `scripts/smoke_playable_slice.sh` 可以复现一次“进入→移动→死亡掉落→拾取”闭环（或撤离闭环）。

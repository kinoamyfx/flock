# Discovery｜市场/竞品（模板 + 待填证据）

## Outcome
形成可复用的竞品对比矩阵与进入策略（先验证，再扩张）。

## Competitor Set（初版）
> 备注：Arcadia 是“经营成长 + 秘境夺宝（Extraction）”的混合体，因此竞品需要拆成“夺宝类/经营类/融合类”三组对照。

### A) 夺宝类（Extraction / 高风险夺宝）
- Escape from Tarkov：典型高风险夺宝、强经济与高门槛；公平性/作弊/审计是核心痛点。
- Dark and Darker：奇幻题材夺宝，合作与 PvP 张力强；强调战斗与撤离读条。
- Dungeonborne / 近似类：同类题材对照，用于验证“规则 + UI 信息层级”是否足够清晰。

### B) 经营类（长线成长 / 资源循环）
- Stardew Valley（星露谷物语）：经营/成长的“舒适感与目标感”标杆；对照“低挫败与反馈”。
- RimWorld：系统深度与事件驱动体验强；对照“可解释性/可回归”与内容扩展。

### C) 融合类（经营/据点 + 冒险）
- Moonlighter：经营（店铺）+ 地牢循环的清晰结构；对照“准备→冒险→结算”节奏与 UI 的任务感。
- Cult of the Lamb：基地经营 + 地牢推进；对照“短周期回合”的节奏设计与奖励反馈。

## Channel Hypotheses（渠道假设）
- Steam：愿望单→转化→评价驱动
- 社区内容：短视频/直播展示“秘境夺宝瞬间”
- Mod 社区：表现层 mod 扩展带来长尾

## Go-to-Market（最小进入策略）
1) Demo：只验证“可玩闭环 + 公平可信 + 画风统一”
2) 小规模测试：观察完成率/挫败点/纠纷点
3) 扩张：增加内容与数值深度
